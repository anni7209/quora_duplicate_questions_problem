# quora_duplicate_questions_problem #

### To use the code. ###
- Download the train.csv.zip and test.csv.zip from [Quora dataset](https://www.kaggle.com/c/quora-question-pairs/data)
- Extract them into "input" folder

### directory structure: ###
```
quora_notebook.ipynb
input/train.csv
input/test.csv
```
